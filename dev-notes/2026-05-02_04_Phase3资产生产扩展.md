## P3-1：网络搜索动态入库（commit `1daf783`）

### 新增模块 `backend/art_asset_searcher.py`

统一的资产网络搜索与入库模块，支持三个来源：

#### Pexels（高质量摄影图片）
- 需配置 `PEXELS_API_KEY`（pexels.com 免费注册）
- `search_pexels(query, per_page)` → 搜索 → 下载 JPG → 入库
- 文件保存至 `G_ArtRes/photos/pexels/{photo_id}.jpg`

#### unDraw（SVG 插画，无需 Key）
- 预置 `_UNDRAW_SLUGS` 映射常用关键词（empty-state / success / error / dashboard 等）
- 从 unDraw CDN 下载 SVG，保存至 `G_ArtRes/illustrations/undraw/{slug}.svg`
- API Key 缺失时自动降级到 CDN 直取

#### Poly Haven（PBR 材质 / HDRI，无需 Key）
- `fetch_polyhaven_texture(asset_id, resolution)` 精确下载指定资产
- 从 `https://api.polyhaven.com/files/{id}` 获取文件列表，优先取 PNG diffuse
- 保存至 `G_ArtRes/3d/textures/{id}_{resolution}.png`

#### 统一入库流程
所有来源搜索到资产后，统一执行：
1. 下载文件到 G_ArtRes 对应目录
2. `_upsert_art_asset()` 写入 `art_assets` 表
3. `_update_manifest()` 追加到 `G_ArtRes/manifest.json`

#### 统一搜索入口 + API 端点

```python
# art_asset_searcher.py
async def search_and_store(query, asset_type, limit) -> List[str]:
    if asset_type == "photo":       return await search_pexels(query, limit)
    elif asset_type == "illustration": aid = await fetch_undraw(query); return [aid] if aid else []
    elif asset_type == "texture":   aid = await fetch_polyhaven_texture(query); return [aid] if aid else []
```

```
POST /api/art-assets/search-web
Body: { "query": "mountain", "asset_type": "photo", "limit": 5 }
返回: { "stored": N, "asset_ids": [...] }
```

## P3-4：AI 生图结果自动入库

### 修改 `backend/image_processor.py`

`process_image_request()` 在图片下载完成后，调用新增的 `_store_generated_image()`：

```python
# 下载完成后自动入库
if result_path:
    await _store_generated_image(req_id, req, result_path, image_url)
```

`_store_generated_image()` 做两件事：
1. 写入 `art_assets` 表（source=`ai_generated`，tags 从 prompt 提取关键词）
2. 追加到 `manifest.json`（id 用 req_id，方便去重）

**效果**：同一个 prompt（或语义相近的 prompt）第二次请求时，先查本地库命中就直接返回，不再调用 LightAI API，节省时间和配额。

## 资产入库优先级链（完整版）

```
资产请求（任意 Agent）
    ↓
① 查 art_assets 本地库（DB，命中直接复用）
    ↓ 未命中
② 网络搜索（按类型路由）
   photo        → Pexels API（需 Key）
   illustration → unDraw CDN（无需 Key）
   texture/hdri → Poly Haven API（无需 Key）
   icon         → Iconify（本地库已有 27,784 个，通常命中）
    ↓ 未找到 / API 未配置
③ AI 生图（LightAI 异步，占位先行）
   完成后自动入库 → 下次相同 prompt 直接命中
    ↓ API Key 未配置 / 超时
④ CSS/SVG 代码生成（简单几何/渐变）
⑤ 占位兜底（纯色块 + 文字标注）
```
