## 修复记录（首次运行的 6 个失败）

| 问题 | 根因 | 修复 |
|------|------|------|
| `NameError: actions` | 测试内 `import` 后用模块路径调用 | 改为 `from actions.executor import` |
| `AssertionError: tuple != str` | Mock `execute` 返回了 `(str, None)` tuple | 改为只返回 `str` |
| P4 `no active connection` | `from database import db` 只绑定一次，monkeypatch 不够 | `patched_db` fixture 同步 patch 子模块引用 |
| P3 API `no active connection` | `api/permissions.py` 模块级 db 绑定 | `patched_db` 加 `api.permissions.db` patch |
| P1 blocking patch 不生效 | executor 内部 import hook_registry | 改 patch `hooks.registry.hook_registry` |
| P3 TestClient 事件循环冲突 | sync TestClient + async fixture 不同事件循环 | 改用 `httpx.AsyncClient + ASGITransport` |
