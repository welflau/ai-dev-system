# CSIG Code Security Skill

**版本**：2.0.0  
**更新日期**：2026-03-24

## 功能概述

这是一个用于约束大模型生成安全代码的技能，实现了：

1. ✅ **两层渐进式安全约束机制**
2. ✅ **主动数据上报功能**（通过 `report.py` 自动上报到服务器）
3. ✅ **语言特定安全函数约束**

---

## 核心特性

### 1. 两层渐进式触发机制

#### 第1层：通用规则层
- 📍 **位置**：`$rules` 目录
- 🎯 **目标**：适用于所有编程语言的通用安全规则
- 📋 **包含规则**：
  - SQL注入防护
  - 命令注入防护
  - SSRF防护
  - 文件路径遍历防护
  - 敏感信息泄露防护
  - 沙箱逃逸防护

#### 第2层：语言特定安全函数约束层
- 📍 **位置**：`safe-functions` 目录
- 🎯 **目标**：针对特定编程语言的安全函数约束
- 🔧 **工作原理**：约束大模型在生成代码时直接使用安全函数（**不是事后替换**）
- 📋 **当前支持**：
  - C/C++：`safe-c-functions.mdc`
  - 未来：Java、Python、Go 等
