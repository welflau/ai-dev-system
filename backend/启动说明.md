## 🔧 主要 API 端点

### 系统状态
- `GET /health` - 健康检查，查看可用工具数量
- `GET /tools` - 列出所有已注册的工具（8 个）

### 项目管理
- `POST /api/process` - 提交新的开发需求
- `GET /api/projects` - 获取所有项目列表
- `GET /api/projects/{id}/state` - 获取项目详情
- `POST /api/projects/{id}/execute` - 执行下一个任务
- `POST /api/projects/{id}/execute-all` - 一键全量执行
- `GET /api/projects/{id}/events` - SSE 实时推送
- `GET /api/projects/{id}/download` - ZIP 打包下载

### 文件管理
- `GET /api/projects/{id}/files` - 列出项目文件
- `GET /api/projects/{id}/files/{path}` - 读取文件内容

### LLM 配置
- `GET /api/llm/status` - LLM 配置状态
- `POST /api/llm/test` - 测试 LLM 连接
- `POST /api/llm/config` - 运行时更新 LLM 配置

## 📝 快速测试

### 方法 1：前端界面（推荐）
1. 打开 http://localhost:8000/app
2. 点击"提交需求" → 输入需求 → 提交
3. 点击"一键执行" → 查看 Pipeline 看板实时进度
4. 浏览生成的文件 + 下载 ZIP

### 方法 2：API 文档
1. 打开 http://localhost:8000/docs
2. 点击任意 API 端点 → "Try it out" → "Execute"

### 方法 3：命令行
```powershell
# 健康检查
curl http://localhost:8000/health

# 提交需求
$body = @{ description = "我想开发一个待办事项应用"; tech_stack = @{ backend = "Python"; framework = "FastAPI" } } | ConvertTo-Json -Depth 3
Invoke-RestMethod -Uri "http://localhost:8000/api/process" -Method Post -Body $body -ContentType "application/json"
```
