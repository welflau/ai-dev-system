## 🔧 当前可用的 API 端点

### 1. 系统状态
- `GET /` - 系统根路径，返回版本信息
- `GET /health` - 健康检查，查看可用工具数量
- `GET /tools` - 列出所有已注册的工具

### 2. 项目管理
- `POST /api/process` - 提交新的开发需求
- `GET /api/projects/{project_id}/state` - 获取项目状态
- `POST /api/projects/{project_id}/execute` - 执行项目计划

## 🛠️ 当前已集成的工具

系统已经注册了以下 8 个工具：

## 📝 快速测试

### 方法 1：使用交互式文档（最简单）
1. 打开 http://localhost:8000/docs
2. 点击任意一个 API 端点（如 `GET /`）
3. 点击 "Try it out" 按钮
4. 点击 "Execute" 按钮发送请求
5. 查看响应结果

### 方法 2：使用 curl 命令
```bash
# 测试根路径
curl http://localhost:8000/

# 查看健康状态
curl http://localhost:8000/health

# 查看可用工具
curl http://localhost:8000/tools
```

### 方法 3：提交一个开发需求
```bash
curl -X POST "http://localhost:8000/api/process" \
  -H "Content-Type: application/json" \
  -d '{
    "user_input": "我想开发一个简单的待办事项应用",
    "tech_stack": ["Python", "FastAPI"]
  }'
```
