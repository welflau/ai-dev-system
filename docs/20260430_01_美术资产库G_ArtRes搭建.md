### 1.2 manifest.json 格式

每个资产条目包含：`id`（AST-xxxxxx）、`name`、`path`（仓库相对路径）、`type`、`style`、`format`、`width/height`、`tags`（JSON 数组）、`source`、`source_ref`、`source_url`、`used_count`、`added_at`。

DB 表可从 manifest.json 随时重建，manifest 是权威来源。

## 2. 图标库（Iconify）

### 2.1 导入方式

```bash
cd G_ArtRes
npm install @iconify/json   # ~70MB，200,000+ 图标原始数据
python scripts/import_icons.py
```

脚本从 `node_modules/@iconify/json/json/{set}.json` 读取图标数据，将每个图标的 `body` 包装为完整 SVG 文件输出到对应目录，同时写入 `manifest.json`。

### 2.2 导入的图标集

| 图标集 | 数量 | 风格 | 适用场景 |
|---|---|---|---|
| lucide | 1,759 | outline | Web/App 通用首选 |
| tabler | 6,140 | outline | 全场景覆盖 |
| material-symbols | 16,183 | outline/filled/rounded | Material Design 项目 |
| simple-icons | 3,702 | filled | 品牌 Logo 专用 |

> phosphor 图标集不在 `@iconify/json` 包内，已从配置中移除。

## 4. Poly Haven HDRI

### 4.1 下载方式

Poly Haven 提供免费 REST API，无需密钥：

```
GET https://api.polyhaven.com/info/{asset_id}
GET https://api.polyhaven.com/files/{asset_id}  →  含各分辨率下载 URL
```

请求需设置 User-Agent，否则返回 403。

### 4.2 精选列表（15 个）

| 类别 | 资产 ID |
|---|---|
| 户外日光 | autumn_field_puresky / sunflowers_puresky / goegap / misty_pines |

均为 1K 分辨率（3840×1920 .hdr），约 3-8MB/个，适合游戏和实时渲染。

## 5. Design Tokens

在 `design-tokens/` 下创建三个标准 JSON 文件，遵循 W3C Design Token Community Group 格式规范：

| 文件 | 内容 |
|---|---|
| `colors.json` | primary(10级) / neutral(12级) / success/warning/error / semantic 语义色 |
| `typography.json` | font-family / size(12级) / weight / line-height / letter-spacing / text-style 预设 |
| `spacing.json` | spacing(17级) / border-radius / shadow / z-index / breakpoint |
