## 一、快速验证清单

| # | 功能 | 如何触发 | 预期现象 | 通过标志 |
|---|------|---------|---------|---------|
| 1 | Rules 注入 | 发送"用 camelCase 写一个函数" | AI 回复中出现"应使用 snake_case"类拒绝或纠正 | AI 遵循 global.md 约定 |
| 2 | Diminishing Returns | 让 AI 循环调用同一工具（如 grep 同一关键词 3 次） | 第 3 次后 Budget 报"原地踏步"并停止 | 日志出现"检测到原地踏步" |
| 3 | Compaction | 在一个 session 发送 20+ 条消息 | 日志出现"对话历史已压缩" | 响应速度无明显下降 |
| 4 | Memory 主动注入 | 保存一条 memory → 新 session 问相关问题 | AI 在首屏 system prompt 尾部引用该记忆 | 响应包含记忆内容 |
| 5 | nudge_hook | AI 完成一轮回复（项目内） | 8 秒内聊天面板底部出现蓝色"💡 还有 N 个需求..." | SSE assistant_nudge 事件 |
| 6 | 费用显示 | 发送一条对话 | 顶栏"💰 今日"数值增加 | 大于 $0.0000 |
| 7 | Feature Flags | 发送"关掉 compaction" | AI 调用 set_session_flag，回复"✅ 已设置 compaction = False" | 后续对话不再压缩 |
| 8 | 并行子任务 | 发送"同时创建三个并行子任务：前端/后端/测试" | AI 调用 dispatch_parallel_subtasks，日志出现"⚡ 子任务并行" ×3 | 3 个工单同时进入 PENDING |

### 2.2 P0-2 Diminishing Returns（L1）

**观测位置**：`server.log`

```bash
tail -f server.log | grep -i "原地踏步\|diminishing"
```

**触发方式**：
在 AI 聊天中输入指令让它重复搜索同一个关键字（可直接问"用 grep 搜索 main.py 三次看看"）。

**预期日志**：
```
[budget] WARNING 检测到原地踏步（连续 3 轮每轮增量 [82, 91, 78] < 500 token）
[engine] INFO BudgetExceeded: 检测到原地踏步...
```

**数据库验证**：
```sql
SELECT reason FROM llm_conversations
WHERE reason LIKE '%原地踏步%' LIMIT 3;
```

### 2.3 P1 Prompt Cache 分区（L3）

**观测位置**：Anthropic API 返回的 `cache_read_input_tokens` 字段

```bash
grep -i "cache_read\|cache_hit" server.log | tail -20
```

**预期日志**（第二次及后续对话）：
```
[llm] INFO cache_read_input_tokens=3241, cache_creation_input_tokens=0
```

第一次必然 miss（创建缓存），第二次同一 session 应命中。

**系统提示分区验证**：
```python
# 可在 Python 交互中运行
from agents.chat_assistant import ChatAssistantAgent
agent = ChatAssistantAgent()
import asyncio
prompt = asyncio.run(agent._build_system_prompt({"id":"test","name":"test","description":""},{"traits":"[]","requirements_summary":"","file_tree":""}))
print("CACHE_BOUNDARY" in prompt, prompt.index("<!--CACHE_BOUNDARY-->"))
```

### 2.4 P1 Compaction 对话压缩（L4）

**触发条件**：单 session 历史超过 20 条

**观测位置**：`server.log`

```bash
tail -f server.log | grep -i "compaction\|对话历史\|压缩"
```

**预期日志**：
```
[chat_assistant] INFO 对话历史已压缩（原 23 条 → 摘要+10 条），摘要 450 chars
```

**数据库验证**：
```sql
SELECT session_id, COUNT(*) as msg_count
FROM chat_messages
GROUP BY session_id
ORDER BY msg_count DESC LIMIT 5;
```

超过 20 条的 session 在后端处理时会看到压缩日志。

## 三、自动化验证脚本

```bash
# 运行一键冒烟测试（检查关键指标）
cd backend
python - <<'EOF'
import asyncio, sys
sys.path.insert(0, '.')

async def smoke_test():
    results = []

    # 1. Rules 注入
    from agents.chat_assistant import ChatAssistantAgent
    agent = ChatAssistantAgent()
    prompt = await agent._build_system_prompt(
        {"id":"t","name":"t","description":""},
        {"traits":"[]","requirements_summary":"","file_tree":""}
    )
    ok1 = "全局编码准则" in prompt and "<!--CACHE_BOUNDARY-->" in prompt
    results.append(("Rules注入 + Cache分区", ok1))

    # 2. Diminishing Returns
    from query_engine.budget import Budget
    b = Budget(max_tokens=10000, max_turns=50)
    b.consume(tokens=100, turns=1)
    b.consume(tokens=100, turns=1)
    b.consume(tokens=100, turns=1)
    ok2 = b.is_diminishing()
    results.append(("Diminishing Returns", ok2))

    # 3. Session Flags
    from actions.chat.set_session_flag import get_session_flag, _SESSION_FLAGS
    _SESSION_FLAGS["smoke"] = {"compaction": False}
    ok3 = get_session_flag("smoke", "compaction") == False
    ok3b = get_session_flag("smoke", "nudge") == True  # default
    results.append(("Feature Flags", ok3 and ok3b))

    # 4. DispatchParallelSubtasksAction 注册
    found = any(c.__name__ == "DispatchParallelSubtasksAction" for c in ChatAssistantAgent.action_classes)
    results.append(("并行子任务Action注册", found))

    # 5. DB cost_usd 列
    from database import db
    await db.init_tables()
    cols = await db.fetch_all("PRAGMA table_info(llm_conversations)")
    has_cost = any(c["name"] == "cost_usd" for c in cols)
    results.append(("cost_usd列存在", has_cost))

    print("\n=== A 方向冒烟测试 ===")
    all_pass = True
    for name, ok in results:
        status = "✅ PASS" if ok else "❌ FAIL"
        print(f"  {status}  {name}")
        if not ok:
            all_pass = False
    print(f"\n{'全部通过 🎉' if all_pass else '有失败项，请检查上方'}")
    return all_pass

asyncio.run(smoke_test())
EOF
```
