# AI 自动开发系统 - 快速演示指南

## 🎉 系统已就绪！

你现在可以立即体验 AI 自动开发系统的 MVP 版本。

## 📋 演示步骤

### 第一步：启动后端服务

如果后端服务还没有运行，打开一个新的终端窗口，执行：

```bash
cd D:\A_Works\ai-dev-system\backend
python -m uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```

看到以下信息表示启动成功：
```
INFO:     Uvicorn running on http://0.0.0.0:8000 (Press CTRL+C to quit)
```

### 第二步：打开前端界面

浏览器应该已经自动打开了前端页面。如果没有，请手动打开：

```
D:\A_Works\ai-dev-system\frontend\index.html
```

或者直接双击 `index.html` 文件。

### 第三步：体验系统功能

#### 1. 查看系统状态
- 首页会显示系统概览和统计信息
- 查看已注册的工具数量

#### 2. 提交一个开发需求
- 点击左侧菜单的 "📝 提交需求"
- 在"项目描述"中输入你的需求，例如：
  ```
  我想开发一个简单的待办事项应用，包含以下功能：
  1. 添加新的待办事项
  2. 标记事项为完成状态
  3. 删除已完成的事项
  4. 查看所有待办事项列表
  ```
- 在"技术栈偏好"中输入（可选）：
  ```
  Python, FastAPI, HTML
  ```
- 在"项目名称"中输入（可选）：
  ```
  todo-app
  ```
- 点击"提交需求"按钮

#### 3. 查看项目状态
- 提交成功后会自动跳转到项目看板
- 点击"🔄 刷新项目列表"查看最新状态
- 点击"查看详情"可以看到项目的详细信息

#### 4. 查看可用工具
- 点击左侧菜单的 "🛠️ 工具列表"
- 查看所有已注册的工具及其描述

## 🔧 技术架构

### 后端
- **框架**: FastAPI 0.135.1
- **语言**: Python 3.14
- **工具集成**: 8 个工具（文件操作 + Git 操作）
- **API 文档**: http://localhost:8000/docs

### 前端
- **技术**: 纯 HTML5 + CSS3 + JavaScript (ES6+)
- **风格**: 简洁、现代化的 UI 设计
- **交互**: 响应式、用户友好的界面

## 🚀 下一步开发计划

根据 ROADMAP，下一步将实现：
1. **Task Decomposer（任务分解器）** - 自动分解用户需求为可执行任务
2. **State Manager（状态管理器）** - 管理项目和任务的状态
3. **Orchestrator（协调器）** - 协调多个 Agent 的工作流程
4. **更多 Agent** - ArchitectAgent, DevAgent, TestAgent 等

## 📝 注意事项

1. **后端服务必须先启动**，否则前端无法与后端通信
2. **端口冲突**: 如果 8000 端口被占用，需要修改端口配置
3. **浏览器兼容性**: 建议使用 Chrome、Firefox 或 Edge
4. **CORS 配置**: 后端已配置允许跨域，如果遇到问题请检查网络设置

## 🎯 演示效果

当你提交需求后，系统会：
1. 接收并验证你的需求
2. 生成一个唯一的项目 ID
3. 保存项目状态到内存中
4. 返回项目 ID 给前端

目前系统还处于早期阶段，真正的需求分析和任务分解功能正在开发中。但你可以体验整个用户交互流程！
