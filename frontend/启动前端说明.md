# 前端启动说明

## 快速启动

### 1. 确保后端服务正在运行

后端服务应该已经运行在 `http://localhost:8000`。如果没有运行，请先启动后端：

```bash
cd D:\A_Works\ai-dev-system\backend
python -m uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```

### 2. 打开前端页面

直接用浏览器打开 `index.html` 文件：

```
D:\A_Works\ai-dev-system\frontend\index.html
```

或者双击 `index.html` 文件即可。

## 功能说明

### 首页
- 显示系统概览和统计数据
- 快速开始新项目

### 提交需求
- 输入项目描述（必需）
- 选择技术栈偏好（可选）
- 设置项目名称（可选）
- 提交后系统会自动分析需求并创建项目

### 项目看板
- 查看所有项目列表
- 查看项目状态
- 查看项目详情

### 工具列表
- 查看所有已注册的工具
- 了解工具功能

## 当前已注册的工具

1. **file_writer** - 文件写入工具
2. **file_reader** - 文件读取工具
3. **directory_lister** - 目录列表工具
4. **git_init** - Git 初始化工具
5. **git_add** - Git 添加文件工具
6. **git_commit** - Git 提交工具
7. **git_push** - Git 推送工具
8. **git_create_branch** - Git 创建分支工具

## 注意事项

1. **跨域问题**：如果遇到 CORS 错误，请确保后端已配置允许跨域
2. **API 地址**：确保后端服务运行在 `http://localhost:8000`
3. **浏览器兼容性**：建议使用 Chrome、Firefox 或 Edge 浏览器

## 技术栈
