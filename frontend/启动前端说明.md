# 前端启动说明

## 快速启动

### 1. 确保后端服务正在运行

后端服务应该已经运行在 `http://localhost:8000`。如果没有运行，请先启动后端：

```bash
cd D:\A_Works\ai-dev-system\backend
python main.py
```

### 2. 打开前端页面

浏览器访问：**http://localhost:8000/app**

前端通过 FastAPI 静态文件托管，无需单独启动。

## 功能说明

### 首页
- 系统概览和统计数据
- LLM 连接状态实时显示
- 快速开始新项目

### 提交需求
- 输入项目描述（必需）
- 选择技术栈偏好（可选）
- 设置项目名称（可选）
- 提交后系统自动分析需求并创建项目

### 项目看板
- 查看所有项目列表
- 进度条显示完成百分比
- 点击查看项目详情

### 项目详情 + Pipeline 看板
- **Pipeline 看板**: 横向 5 阶段（需求→架构→开发→测试→部署）+ 连接箭头
- **阶段状态颜色**: 绿色(完成) / 蓝色脉冲(运行中) / 红色(失败) / 灰色(待执行)
- **一键执行**: ⚡ 按钮自动运行所有任务
- **日志抽屉**: 点击步骤查看执行日志（VS Code 风格暗色终端）
- **文件浏览器**: 查看 Agent 生成的全部文件
- **代码高亮**: Python/JS/HTML/CSS/JSON 语法高亮预览
- **SSE 实时更新**: 任务状态自动刷新 + Toast 通知

### 项目下载
- 📦 一键 ZIP 打包下载所有生成的项目文件

### LLM 配置
- 配置 API Base URL / API Key / 模型名称
- 一键测试连接
- 不配置 LLM 也能使用（模板降级模式）

## 技术栈

- 纯 HTML5 + CSS3 + JavaScript（ES6+）
- highlight.js (CDN) 代码语法高亮
- EventSource (SSE 实时更新)
- 响应式设计
- RESTful API 交互

## 文件结构

```
frontend/
├── index.html    # 主页面 (全部 HTML + CSS)
└── app.js        # 应用逻辑 (路由/API/SSE/Pipeline看板/高亮)
```

## 注意事项

1. **无需 Node.js**: 前端为纯原生实现，不需要 npm install
2. **无需单独启动**: 前端通过后端 `/app` 路由托管
3. **跨域**: 后端已配置 CORS
4. **浏览器兼容性**: 建议使用 Chrome、Firefox 或 Edge
